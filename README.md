## Feature & 功能亮点 <!-- {docsify-ignore-all} -->

- 可以在线编辑 `html` | `css` | `js` 代码, 并实时预览代码效果
- 代码可以自动保存到页面的 URL 上, 分享网址即可分享代码
- 静态资源全局缓存, 多个 sandbox 之间资源共享, 拒绝浪费请求
